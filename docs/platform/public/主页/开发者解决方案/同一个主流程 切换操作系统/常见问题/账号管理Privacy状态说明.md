# 账号管理里的 Privacy / Fail 是什么意思？

在管理后台的账号管理列表中，OpenAI OAuth 账号可能会显示 `Private`、`Fail` 或 `CF` 标签。这些标签表示系统尝试关闭 ChatGPT 训练数据共享设置的结果，不等同于账号是否可用。

## 状态含义

- `Private`：已确认关闭训练数据共享，隐私状态较理想。
- `Fail`：关闭训练数据共享的自动请求失败，状态不确定，可能仍开启。
- `CF`：请求被 Cloudflare 拦截，状态不确定，可能仍开启。
- 无标签：未记录过隐私设置结果，或该账号类型不支持此状态展示。

## 是否影响使用？

通常不直接影响 API 转发、额度消耗和调度。账号是否参与调度，主要看账号列表中的“状态”和“调度”开关。

例如：

- `Fail` + 状态“正常” + 调度开启：账号仍可能被使用。
- `Fail` + 状态“暂停”或调度关闭：账号不会正常参与调度。

## 如何处理 Fail？

在账号右侧点击“更多”，选择“设置隐私”重试。

如果仍然失败，建议先执行：

1. 刷新 Token。
2. 或重新授权账号。
3. 然后再次点击“设置隐私”。

如果多次失败，通常需要检查账号 Token、代理网络、上游接口返回或 Cloudflare 拦截情况。

## 二、功能区文案

### 官方直连

只做明确来源的 OpenAI 官方 API 接入，不把订阅产品、混合来源和模型扣费口径混在一起说明。

### 透明计费

模型价格、充值规则、余额变化和使用记录分开说明。每次调用后，可在站内查看模型、Token 和费用记录。

### 快速接入

兼容 OpenAI 风格 API。创建 API Key 后，将 Base URL 替换为本站地址，即可用常见 SDK 或工具开始测试。

### 用量可控

支持 API Key 管理、用量记录、额度控制和异常排查，适合个人项目、小团队和内部工具统一接入。

## 三、新手区文案

第一次使用只需要四步：

1. 注册并登录账号。
2. 充值或兑换余额。
3. 创建 API Key。
4. 复制 Base URL 并发起测试请求。

建议先使用轻量模型发起一条文本请求，确认链路跑通后再接入真实项目。

## 四、信任区文案

我们更重视清楚和可核查，而不是夸张承诺：

- 服务来源公开说明。
- 费用口径长期维护。
- 异常处理规则可查看。
- 使用记录和余额变化可查询。

如遇到上游波动、计费疑问或调用异常，请优先查看费用公告、使用指南和售后规则，再通过公开客服入口提交问题。

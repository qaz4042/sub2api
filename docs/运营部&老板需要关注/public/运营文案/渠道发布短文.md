## 一、朋友圈 / 群公告

### 版本 A

我整理了一个 OpenAI API 接入站，适合开发者和小团队使用。

特点是官方直连、计费透明、API Key 可管理、用量记录可查询。支持 OpenAI 风格接口，创建 Key 后复制 Base URL 就能测试。

适合想快速接入 Codex App、OpenAI SDK 或内部工具的用户。第一次使用建议先看费用公告和使用指南。

### 版本 B

Sub2API 现在适合做两类事情：

1. 个人开发者快速接入 OpenAI 风格 API。
2. 小团队统一管理 API Key、额度和用量。

不主打无限量和低价噱头，重点是来源清楚、价格清楚、接入清楚、异常处理清楚。

## 三、README 简短介绍

Sub2API 是面向开发者和小团队的 AI API 网关，支持 OpenAI 风格接口接入、API Key 管理、用量统计、余额扣费、限流和后台管理。适合公开接入站点、团队内部 AI 网关和私有化部署场景。

## 四、私有部署推广

标题：

> 给团队部署一套自己的 AI API 网关

正文：

> 如果团队里多人都在使用 AI API，常见问题是 Key 分散、账单不清、额度不可控、出问题难排查。Sub2API 可以部署到客户自己的服务器或云账号中，统一管理 API Key、模型、额度、用量记录和基础运维配置。
>
> 标准交付包括服务部署、后台初始化、调用测试、备份说明和交接文档。适合 3-30 人开发团队、工作室和已有内部系统的小企业。

## 五、客服引流短句

# 文档目录

本文档按“是否有必要向普通用户公开”整理。

## public

面向普通用户、访客或需要公开说明的内容，可放到站点帮助中心、公告页或公开仓库文档中。

- `使用指南.md`：用户注册、充值、创建 API Key 和调用测试。
- `费用公告.md`：计费口径、充值规则和异常处理说明。
- `开发者解决方案.md`：开发者接入 Codex App / CC Switch 的说明。
- `企业解决方案.md`：对外展示的企业服务说明。
- `DISCLAIMER.md`：项目风险声明。
- `legal/`：可公开链接的合规承诺文本。

## operator

面向部署者、管理员、运维人员或支付对接方。普通用户不需要看到，避免暴露管理接口、部署细节和服务端配置。

- 部署与运维：`DEPLOYMENT.md`、`OPERATIONS.md`、`MODES.md`
- 支付配置：`PAYMENT_CN.md`、`PAYMENT.md`
- 管理接口与平台配置：`ADMIN_PAYMENT_INTEGRATION_API.md`、`平台配置管理.md`

## internal

内部运营判断、商业化策略、上线前检查和实例级配置。默认不公开，避免泄露运营策略、实例状态、域名配置、安全缺口或未确认事项。

- `个人启动与商业化决策摘要.md`
- `站点运营思路.md`
- `站点运营思路_补充.md`
- `运营前配置与数据检查清单.md`
- `运营前待确认配置示例.md`

# 开发者推广文案

更新日期：2026-06-27

适用对象：个人开发者、独立开发者、Codex App 用户、CC Switch 用户。

## 一、核心卖点

推荐表达：

> Sub2API 适合希望快速接入 OpenAI 风格 API 的开发者。创建 API Key，复制 Base URL，即可在支持自定义接口地址的工具或 SDK 中使用。

可展开为：

- 不必反复处理账号、网络、支付和配置细节。
- 支持 OpenAI 风格接口接入。
- 可查看用量记录和余额变化。
- 可为不同项目创建独立 API Key。
- 支持 Codex App 配合 CC Switch 使用。

## 二、场景文案

### Codex App 接入

标题：

> Codex App 接入自定义 API 更简单

正文：

> 使用 Sub2API 创建 API Key 后，可以通过 CC Switch 导入配置，再在 Codex App 中使用。适合希望快速完成开发环境配置、减少反复切换账号和接口地址的用户。

### OpenAI SDK 接入

标题：

> 保留熟悉的 OpenAI SDK 使用方式

正文：

> 如果你的项目已经使用 OpenAI 风格 SDK，通常只需要替换 Base URL，并填入站内生成的 API Key。建议先用简单文本请求验证链路，再接入真实业务。

### 个人项目开发

标题：

> 给个人项目一个清楚的 API 入口

正文：

> 不同项目使用不同 Key，既方便统计费用，也方便出现问题时快速禁用。适合脚本、插件、内部工具、自动化任务和轻量产品原型。

## 三、短版推广文案

### 版本 A

Sub2API 是一个面向开发者的 OpenAI API 网关。支持创建 API Key、复制 Base URL、查看用量和余额记录，适合 Codex App、OpenAI SDK 和各种支持自定义接口地址的工具。

### 版本 B

想快速跑通 OpenAI 风格 API，可以试试 Sub2API：登录后创建 Key，复制 Base URL，先用 curl 跑通第一条请求，再接入自己的项目。

### 版本 C

给开发者准备的 AI API 接入口：官方直连、计费透明、Key 可管理、用量可查询。适合个人项目、小工具和 Codex 开发场景。

## 四、长版推广文案

Sub2API 面向开发者和小团队，提供 OpenAI API 官方直连接入、API Key 管理、余额扣费和用量记录。

如果你只是想快速把 OpenAI 风格 API 接入到工具、脚本或项目里，可以按四步完成：

1. 登录账号。
2. 充值或兑换余额。
3. 创建 API Key。
4. 复制 Base URL 并发起测试请求。

它不主打复杂套餐，也不把充值金额、余额和 Token 价格混在一起解释。建议先看费用公告，再用最小请求跑通链路。

推广时不要承诺：

- 所有模型都支持。
- 永不封禁。
- 无限调用。
- 全网最低价。
- 与 OpenAI 官方存在合作或背书关系。

建议统一使用“以站内模型列表、费用公告和实际使用记录为准”。
